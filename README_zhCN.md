## ✨ 功能

### 素材

5 个字体，6 个材质以及 2 个提示音。

### 物品

1. **删除物品**：删除时自动填入“DELETE”，支持点击确认和 Delete 键确认，并可自定义确认按钮风格。
2. **已学配方染色**：为已学配方、玩具、战宠和坐骑添加彩色遮罩；支持自定义颜色与透明度，适用于背包、邮箱与商人页。
3. **快速拾取**：大大加速自动拾取战利品的速度。
4. **交易**：在交易界面添加一键感谢按钮，快速向对方致谢。
5. **额外物品条**：为药水、药剂、任务物品及自定义物品创建可配置的物品条，最多支持 5 个条，按钮大小、间距、背景和槽位数量均可设置，并支持自动显隐与快捷键。
6. **联系人**：在邮箱旁新增联系人面板，支持收藏夹与快速邮寄。
7. **装备观察**：在角色面板与观察界面旁显示装备列表，展示全部栏位、物品等级、宝石与附魔信息。
8. **商人页面扩展**：拓展商人界面，一次显示更多物品，提升浏览效率。

### 战斗

1. **团队标记**：提供可自定义的标记条，包含世界标记、骷髅/叉、准备就绪确认、拉怪倒计时与清除标记；可调按钮大小、间距、背景、快捷键与显示条件（队伍/团队/单人）。
2. **战斗提醒**：在进入或离开战斗时播放可自定义的动画与屏幕效果，支持尺寸、速度与颜色调节。
3. **快速钥石**：打开史诗钥石界面时自动插入背包中的钥石。
4. **职业助手**：为每个职业提供专属工具：死亡骑士（灵打治疗量预估）。

### 地图

1. **超级追踪**：为路径点提供自动追踪、中键清除、移除距离限制、命令解析以及虚拟 TomTom 指令等功能。
2. **小地图点击者**：显示点击小地图的玩家姓名，并可自定义动画样式与颜色。
3. **矩形小地图**：让小地图的长宽独立设置，打造矩形外观。
4. **小地图按钮**：将所有小地图按钮集中到一条按钮条中，可自定义布局、排序以及版本大厅按钮支持。
5. **世界地图**：移除世界地图迷雾或自定义迷雾颜色，可根据需要缩放地图。
6. **副本难度**：以文字样式重新美化副本难度指示，并可自定义字体与文本。
7. **事件追踪器**：在世界地图跟踪 TWW 周常、夜幕、剧团、光耀回响、巨龙群岛事件等世界事件，并提供可配置的提醒与音效。

### 任务

1. **任务列表**：自定义任务追踪标题的字体、颜色与装饰条，支持彩色进度文本、百分比显示与独立背景。
2. **交接**：自动接受与提交任务，智能选择奖励，可配置暂停修饰键，并支持暗月马戏团、随从委派等特殊对话及 NPC 黑名单。
3. **切换按钮**：提供可自定义的小按钮条，以便快速启用或关闭交接与通告模块。

### 社交

1. **聊天条**：提供频道切换条，支持自定义世界/社区频道、表情面板、掷骰子与按钮别名，可切换水平或垂直布局、鼠标悬停/常驻样式与按钮尺寸。
2. **聊天链接**：为物品、法术等链接添加图标、等级、品质数值、护甲/武器类别以及名称翻译，可自定义图标大小。
3. **聊天文字**：提供 8 套职责图标（FFXIV、PhilMod、Hexagon、ElvUI、SunUI、LynUI、ElvUI Old、Blizzard），支持去除括号与服务器、添加职业/阵营图标、优化公会成员状态、合并成就与战网上下线提示，并可自定义频道缩写。
4. **右键菜单**：无污染菜单，新增公会邀请、查询、属性报告与英雄榜（含跨区设置）等选项，并带有分组标题样式。
5. **表情**：解析并发送表情，支持自定义图标大小，可通过“{”键打开面板，并兼容聊天泡泡。
6. **好友名单**：支持（默认/暗黑/方块）风格状态图标，游戏图标样式，彩色等级（可隐藏满级），职业/客户端染色，使用备注替代名字，隐藏服务器，以及自定义姓名与信息字体。
7. **智能 Tab**：使用 Tab 智能切换聊天频道，可配置是否包含喊话、战场、团队通告与官员频道，并记录跨角色私聊历史及过期时间。

### 通告

1. **任务**：向队友通报任务进度。与此同时，提供比暴雪更好看的彩色任务进度信息。
2. **打断**：打断提示支持自定义文本，可识别宠物技能并显示被打断的法术。
3. **驱散**：在成功驱散时发送提示，文本可自定义。
4. **嘲讽**：嘲讽提示支持宠物与武僧雕像等情况，并可自定义信息和目标展示。
5. **战斗复活**：通报战复使用情况，可选择频道（队伍/团队/团队警告）。
6. **实用技能**：针对机器人、盛宴、传送门、玩具等技能施放时自动发送提示，文本可配置。
7. **仇恨转移**：在使用威胁转移技能时通告目标信息。
8. **告别**：副本或团队结束后自动发送告别语，可设置延迟与文本。
9. **感谢**：感谢复活、增益等帮助的玩家，支持延迟和触发条件。
10. **副本重置**：重置副本后提醒队友。
11. **钥石**：通报大秘境钥石信息，并提供自定义文本格式。

### 鼠标提示

1. **通用**: 可自定义修饰键; 为物品、法术、成就添加标题图标; 阵营图标; 宠物图标和 ID; 套装信息
2. **ElvUI 鼠标提示调整**: 调整生命条和文本 Y 轴偏移; 添加专精图标和种族图标并支持自定义大小; 增强大秘境信息并添加地下城图标; 强制显示物品等级选项
3. **进度**: 显示团本进度、特殊成就和大秘境评分，支持自定义标题样式 (无/文字/材质); 支持副本筛选和最高分标记; 战斗中禁用选项
4. **钥石**: 在鼠标提示中显示钥石信息 (需对方安装 Details!，BigWigs 或者 WindTools); 支持缩写和自定义图标大小
5. **队伍信息**: 添加详细的预组建队伍信息，支持自定义模板、职业图标样式、紧凑/普通模式; 可选排除地下城和隐藏暴雪默认信息
6. **目标进度**: 为怪物添加详细的目标进度信息，支持可调精度

### 单位框体

1. **快速焦点**：通过组合修饰键与鼠标左右键/中键，在任意单位框体上快速设置焦点。
2. **吸收**：自定义 ElvUI 吸收条材质，并在生命条末端添加暴雪风格的吸收火花，支持调节尺寸、颜色与偏移。
3. **职责图标**：在所有单位框体上应用多套职责图标主题（FFXIV、PhilMod、Hexagon、ElvUI、SunUI、LynUI、ElvUI Old、Blizzard）。

4. **顿号转斜杠**：自动把中文符号（、）转为斜杠“/”，方便输入命令。
5. **关闭特写**：禁用暴雪特写框体。
6. **标签**：为 ElvUI 增加距离、无符号生命/能量、职业色、图标等自定义标签。
7. **游戏条**：现代化系统状态条，提供主页、公会、好友、日历、音量、系统、静音、设置、游戏菜单按钮，可自由放置左右两侧，显示时间/延迟并支持字体、颜色、背景、鼠标提示及战斗模式自动隐藏。
8. **跳过过场动画**：自动跳过大部分过场动画，按下修饰键可暂时禁用，并可设置副本内行为。
9. **预组列表**：为预组队列添加职业颜色条、更大的职业图标与额外成员信息。
10. **移动速度**：在角色面板显示当前移动速度。
11. **自动化**：自动接受复活/召唤/任务、战斗中隐藏部分框体。
12. **无汉字计数**：将缩写中的单位数字替换为英文字母。
13. **自动截图**：在获得成就时自动截图。
14. **隐藏制造者**：隐藏鼠标提示中的制造者姓名。
15. **键位在冷却上方**：让快捷键文字显示在 ElvUI 冷却动画之上，更易识别。
16. **公会新闻装等**：在公会新闻中显示装备物品等级。
17. **自定义快捷键别名**：为快捷键创建自定义显示名称（如将“BUTTON1”显示为“左键”）。
18. **冷却文字偏移**：分别调节动作条、单位框体与姓名板上的冷却文字偏移。
19. **退出相位潜行**：通过按钮快速离开相位潜行区域。
20. **雷什裹布升级**：鼠标中键点击雷什裹布即可打开升级菜单。
