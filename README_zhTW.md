## ✨ 功能

### 素材

5 個字型，6 個材質以及 2 個提示音。

### 物品

1. **刪除物品**：刪除時自動填入「DELETE」，支援點擊確認與 Delete 鍵確認，並可自訂確認按鈕樣式。
2. **已學配方上色**：為已學的配方、玩具、戰寵與坐騎加上彩色遮罩；支援自訂顏色與透明度，適用於背包、郵箱與商人介面。
3. **快速拾取**：大大加速自動拾取戰利品的速度。
4. **交易**：在交易框體新增一鍵感謝按鈕，快速向對方致意。
5. **額外物品條**：建立可自訂的物品條來放置藥水、藥劑、任務物品及自訂物品，最多支援 5 個條，按鍵大小、間距、背景與欄位數皆可設定，並支援自動顯示/隱藏與快捷鍵。
6. **聯絡人**：在郵件介面旁新增聯絡人面板，具備常用名單與快速寄信功能。
7. **裝備觀察**：在角色面板與觀察介面旁顯示裝備列表，列出全部欄位、物品等級、寶石與附魔資訊，並可點擊開啟試衣間。
8. **擴展商人頁面**：擴充商人畫面，一次顯示更多物品，提升購買效率。

### 戰鬥

1. **團隊標記**：提供可自訂的標記列，包含世界標記、骷髏/叉、準備確認、拉怪倒數與清除標記；可調整按鈕尺寸、間距、背景、快捷鍵與顯示條件（隊伍/團隊/單人）。
2. **戰鬥提醒**：進出戰鬥時顯示可自訂的動畫與特效，以及尺寸、速度與顏色調整。
3. **快速鑰石**：打開傳奇鑰石介面時，自動放入背包中的鑰石。
4. **職業助手**：為各職業提供專屬實用功能：死亡騎士（死亡打擊治療量預估）。

### 地圖

1. **超級追蹤**：提供路徑點自動追蹤、滑鼠中鍵清除、移除距離限制、命令解析與虛擬 TomTom 指令等功能。
2. **小地圖點擊者**：顯示點擊小地圖的玩家名稱，並可自訂動畫效果與顏色。
3. **矩形小地圖**：自訂小地圖的寬與高，打造矩形外觀。
4. **小地圖按鍵**：將所有小地圖按鍵集中於按鍵列，支援自訂版面、排序與版本大廳按鍵。
5. **世界地圖**：移除世界地圖迷霧並提供彩色迷霧選項，可自訂縮放比例。
6. **副本難度**：以文字樣式重新美化副本難度圖示，並可設定字型與顯示內容。
7. **事件追蹤器**：在世界地圖顯示 TWW 每週任務、夜幕、劇團、回響、巨龍群島事件等資訊，並具備可調提醒與音效。

### 任務

1. **任務列表**：自訂任務追蹤標題的字型、顏色與裝飾條，支援彩色進度文字、百分比顯示與背景。
2. **交接**：自動接受與完成任務，具備最佳獎勵選擇、可配置暫停修飾鍵，並支援暗月馬戲團、追隨者指派等特殊對話與 NPC 黑名單。
3. **切換按鍵**：提供可自訂的小按鍵列，快速切換交接與通告模組。

### 社交

1. **聊天條**：提供頻道切換列，支援自訂世界/社群頻道、表情面板、擲骰子與按鈕別名，可切換水平或垂直、滑鼠懸停或常駐模式，以及按鍵尺寸。
2. **聊天鏈接**：在物品、法術等連結上顯示圖示、等級、品質數值、護甲/武器分類與翻譯名稱，並可自訂圖示大小。
3. **聊天文字**：提供 8 套職責圖示（FFXIV、PhilMod、Hexagon、ElvUI、SunUI、LynUI、ElvUI Old、Blizzard），支援移除括號與伺服器、加入職業/陣營圖示、強化公會成員狀態、合併成就與戰網上下線通知，並自訂頻道縮寫。
4. **右鍵選單**：無污染右鍵選單，新增公會邀請、/who、屬性匯報與英雄榜（含跨區設定）等選項，並附有分區標題樣式。
5. **表情**：解析並傳送表情，可設定圖示大小，透過「{」鍵開啟面板並支援聊天泡泡。
6. **好友名單**：支援（預設/暗黑/方塊）風格狀態圖示、遊戲圖示風格、彩色等級（可隱藏滿等）、職業/客戶端著色、以備註顯示名稱、隱藏伺服器，並自訂名稱與資訊字型。
7. **智能 Tab**：以 Tab 智慧切換聊天頻道，可設定是否包含大喊、戰場、團隊警告與官員頻道，並保留跨角色密語歷史與到期時間。

### 通告

1. **任務**：向隊友通報任務進度。與此同時，提供比暴雪更好看的彩色任務進度訊息。
2. **打斷**：打斷提示支援自訂訊息，可辨識寵物技能並顯示被打斷的法術。
3. **驅散**：驅散成功時發送提示，訊息可自訂。
4. **嘲諷**：嘲諷通告支援寵物與武僧雕像等情況，可自訂內容並顯示目標。
5. **戰鬥復活**：通報戰復施放情況，可選擇頻道（隊伍/團隊/團隊警告）。
6. **實用技能**：針對機器人、盛宴、傳送門、玩具等分類發送使用通告，訊息可配置。
7. **仇恨轉移**：提示威脅轉移技能的施放並顯示目標資訊。
8. **告別**：副本或團隊結束後自動送上告別訊息，可設定延遲與文字。
9. **感謝**：感謝復活、增益等協助的玩家，可調整延遲與觸發條件。
10. **副本重置**：副本重置後提醒隊友。
11. **鑰石**：通報傳奇鑰石資訊並支援自訂訊息格式。

### 浮動提示

1. **通用**: 可自訂修飾鍵; 為物品、法術、成就新增標題圖示; 陣營圖示; 寵物圖示和 ID; 套裝資訊
2. **ElvUI 浮動提示調整**: 調整生命條和文字 Y 軸偏移; 新增專精圖示和種族圖示並支援自訂大小; 增強傳奇鑰石資訊並新增地城圖示; 強制顯示物品等級選項
3. **進度**: 顯示團本進度、特殊成就和傳奇鑰石評分，支援自訂標題樣式 (無/文字/材質); 支援副本篩選和最高分標記; 戰鬥中禁用選項
4. **鑰石**: 在浮動提示中顯示鑰石資訊 (需要對方安裝 Details!，Bigwigs 或 WindTools); 支援縮寫和自訂圖示大小
5. **隊伍資訊**: 新增詳細的預組隊伍資訊，支援自訂範本、職業圖示樣式、精簡/普通模式; 可選排除地城和隱藏暴雪預設資訊
6. **目標進度**: 為怪物新增詳細的目標進度資訊，支援可調精度

### 單位框架

1. **快速焦點**：透過修飾鍵搭配滑鼠左右鍵/中鍵，在任何單位框架上快速設定焦點。
2. **吸收**：自訂 ElvUI 吸收條材質，並於生命條末端加入暴雪風格的火花效果，可調整尺寸、顏色與偏移。
3. **職責圖標**：在所有單位框架套用多組職責圖示主題（FFXIV、PhilMod、Hexagon、ElvUI、SunUI、LynUI、ElvUI Old、Blizzard）。
